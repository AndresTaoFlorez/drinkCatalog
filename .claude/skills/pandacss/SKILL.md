Apply PandaCSS patterns for this project (jsxFramework: "vue", outdir: "styled-system").

**Imports:**
- Styled components: `import { styled } from "@styled-system/jsx"`
- CSS utility function: `import { css } from "@styled-system/css"`
- Global CSS (already in main.ts): `import "@/styled-system/index.css"`
- Path alias: `@styled-system` → `./styled-system/`

**Styled components pattern (used in `*.styles.ts` files):**
```ts
import { styled } from "@styled-system/jsx"

export const Card = styled("div", {
  base: {
    display: "flex",
    p: "4",
    borderRadius: "md",
  },
  variants: {
    variant: {
      outline: { border: "1px solid", borderColor: "gray.200" },
      filled: { bg: "gray.100" },
    },
  },
})
```

**Token usage:**
- Use design tokens, not raw CSS values: `p: "4"` not `padding: "16px"`, `color: "gray.700"` not `color: "#374151"`.
- Spacing scale: `"1"` = 4px, `"2"` = 8px, `"4"` = 16px, `"6"` = 24px, `"8"` = 32px, `"14"` = 56px.

**Do not:**
- Edit files inside `styled-system/` — it is auto-generated by `npx panda codegen`.
- Write `<style>` blocks in `.vue` files — all styling goes in `.styles.ts`.
- Use inline `style` bindings for anything a token covers.

**Regenerate after config changes:**
```sh
npx panda codegen
```

**When adding new tokens or theme extensions**, edit `panda.config.ts` under `theme.extend`, then run codegen.
